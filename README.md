# table-reserver
La Maison Restaurant Reservation System is a fully interactive frontend web application that allows guests to browse a live SVG floor plan of the restaurant and select their preferred table across three distinct dining zones — Wine Cellar, Main Dining Room, and Garden Terrace. Once a table is chosen, a smooth reservation form slides into view, collecting guest details, date, time, party size, and special occasion notes, with built-in validation to ensure the guest count never exceeds the table's capacity. The app ships with rich mock data including 16 uniquely positioned tables and 6 pre-loaded reservations, each carrying realistic guest profiles, statuses, and dietary notes. A live reservations dashboard below the floor plan lets restaurant staff search, filter by zone, seat arriving guests, or cancel bookings in real time — with the floor plan updating instantly to reflect availability. Built with pure HTML, CSS, and vanilla JavaScript across 5 clean, modular files, it requires zero dependencies or build tools and runs directly in any browser. The design uses a refined deep-green and gold palette with Playfair Display typography, giving the interface the warmth and elegance expected of a fine-dining establishment.
